Semantics of Sustainability

Exploring conceptual change in sustainability language through machine learning.

About the Project

This project, funded by the NL eScience Center, uses machine learning techniques to study conceptual change over time. It builds on the BERT infrastructure to analyze historical Dutch textual data, showing how words related to sustainability have evolved from WWII to today.

By retraining the base model on Dutch National Library data, the project reveals shifts in meaning and connotation, contributing to the study of global sustainability's conceptual history.

Collaboration: eScience Center, Utrecht University, National Library The Hague.
