About the Semantics of Sustainability

Exploring the evolution of sustainability-related language using advanced machine learning techniques.

Project Overview

The Semantics of Sustainability project, funded by the NL eScience Center, investigates how the language of sustainability has evolved over time. Through machine learning, we analyze historical Dutch texts to track changes in sustainability discourse from WWII to today.

Goals & Methodology

Data Collection

We gather historical data from a variety of sources, including digitized archives, libraries, and research institutions.

Machine Learning

Using machine learning models like BERT, we analyze the contextual evolution of sustainability-related terms.

Analysis

We visualize the results to show how language has adapted to new sustainability challenges over time.
